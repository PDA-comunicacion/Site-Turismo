# ¿Por qué Pabellón de Arteaga? 🍇🍷

Este proyecto es una plataforma digital turística integral desarrollada para promover la riqueza cultural, histórica y gastronómica del municipio de **Pabellón de Arteaga, Aguascalientes**. El sitio ofrece un recorrido visual e interactivo a través de diversas rutas temáticas, permitiendo a los visitantes descubrir desde viñedos de clase mundial hasta tesoros arquitectónicos históricos.

## 🚀 Características Principales

El sitio web está estructurado en diversas **Rutas Turísticas** interactivas, diseñadas para una navegación intuitiva:

* **Ruta del Enoturismo:** Información detallada sobre las principales vinícolas de la región como *Hacienda de Letras*, *Vinícola Santa Elena* y *Bodegas Origen*.
* **Ruta de la Garnacha:** Un catálogo gastronómico de los rincones más emblemáticos para disfrutar de la cocina local, como *Gorditas Ime* y *Casa Nacho*.
* **Ruta de las Haciendas:** Recorrido histórico por sitios emblemáticos como la *Hacienda El Mezquite* y la *Hacienda de Garabato*.
* **Ruta Holística:** Espacios dedicados al bienestar integral y la relajación, destacando lugares como *Tonalli Spa*.
* **Ruta Fotográfica:** Información sobre concursos activos como "Colores de la Tierra" para capturar la esencia del municipio.
* **Ruta Religiosa:** Exploración de templos y capillas con alto valor histórico, incluyendo el *Templo de Guadalupe* y el *Sagrado Corazón*.

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este portal se emplearon las siguientes herramientas y lenguajes:

* **Frontend:** HTML5, CSS3 y JavaScript para una estructura robusta y diseño personalizado.
* **Bootstrap 5:** Framework utilizado para garantizar un diseño responsivo y componentes modernos.
* **Bibliotecas JS:** * **Flickity:** Para la creación de carruseles de imágenes y galerías interactivas.
    * **jQuery:** Para la manipulación del DOM y efectos de transición suave.
* **Iconografía:** Font Awesome para mejorar la interfaz de usuario.
* **Tipografía:** Google Fonts (*Montserrat*) y fuentes personalizadas (*Coolvetica*).

## 📱 Diseño Responsivo

El sitio cuenta con una arquitectura adaptable mediante `media_queries.css`, asegurando una visualización óptima en:
* Escritorios de alta resolución y laptops.
* Tabletas y dispositivos móviles de diversos tamaños.
* Pantallas extra pequeñas con optimización de menús flotantes.
